Soothing Systems

A set of small P5 projects to provide proof of concepts for Soothing System's generative graphics

  • Breath: Example of sinusoidally eased breathing meter
  • Mandelbrot: An example fractal taken directly from P5's example set.
  • Julia: A fractal with a dynamic control point.
  • Breathing Julia: Varying a fractal's color coding with breath.
  • Camera Julia: mouse gestures control the fractal's parameters.
  • P5 DOM Interaction: An HTML menu interacts with a P5 sketches parameters
